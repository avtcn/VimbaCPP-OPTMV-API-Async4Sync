# Introduction 
Implement OPTMV Camera Interface API based on Vimba SDK C++ API.

# Getting Started
* Vimba SDK
  Use the latest version, such as Vimba 2.1.3 on http://www.alliedvision.com/
* OPTMV Camera API or similar one
  The API includes following calls:
  1. OpenCamera()
  2. StartAcquire()
  3. Shot()
  4. StopAcquire()
  5. CloseCamera()
